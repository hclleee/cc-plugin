# 워크플로우 삭제

`.claude/workflows/` 디렉토리에서 지정된 워크플로우를 삭제합니다.

## 실행 절차

1. **워크플로우 확인**
   - 인자로 워크플로우 이름이 제공되지 않으면:
     - `/workflow-list` 결과를 보여주고
     - 삭제할 워크플로우 이름 입력 요청

2. **파일 존재 확인**
   - `.claude/workflows/<이름>.json` 파일 존재 여부 확인
   - 없으면 "워크플로우 '<이름>'을 찾을 수 없습니다." 메시지 출력

3. **삭제 확인**
   - AskUserQuestion 도구로 삭제 확인:
     ```
     워크플로우 '<이름>'을 삭제하시겠습니까?
     - 삭제: 영구적으로 삭제합니다
     - 취소: 삭제를 취소합니다
     ```

4. **삭제 실행**
   - 사용자가 "삭제" 선택 시 파일 삭제
   - 성공 메시지 출력

## 사용 예시

```
/workflow-delete code-review
```

## 출력 예시

```
## 워크플로우 삭제

워크플로우 'code-review'를 삭제하시겠습니까?

이 작업은 되돌릴 수 없습니다.
파일: .claude/workflows/code-review.json

[삭제 확인 후]

워크플로우 'code-review'가 삭제되었습니다.
```

## 주의사항

- 삭제된 워크플로우는 복구할 수 없습니다
- Git으로 관리 중이라면 `git checkout`으로 복구 가능
