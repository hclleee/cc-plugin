# 워크플로우 목록 조회

`.claude/workflows/` 디렉토리에 저장된 모든 워크플로우를 조회합니다.

## 실행 절차

1. **디렉토리 확인**
   - `.claude/workflows/` 디렉토리 존재 여부 확인
   - 없으면 "워크플로우가 없습니다. `/workflow-create`로 새 워크플로우를 생성하세요." 메시지 출력

2. **워크플로우 파일 스캔**
   - `*.json` 파일들을 검색
   - 각 파일에서 `name`, `description`, `version`, `updatedAt` 추출

3. **목록 출력**
   - 테이블 형식으로 출력:

   ```
   | # | 이름 | 설명 | 버전 | 최종 수정 |
   |---|------|------|------|-----------|
   | 1 | code-review | 코드 리뷰 워크플로우 | 1.0.0 | 2025-01-06 |
   | 2 | deploy-check | 배포 전 점검 | 1.2.0 | 2025-01-05 |
   ```

4. **추가 안내**
   - 워크플로우 실행: `/workflow-run <이름>`
   - 워크플로우 편집: `/workflow-edit <이름>`
   - 워크플로우 삭제: `/workflow-delete <이름>`
   - 새 워크플로우 생성: `/workflow-create`

## 출력 예시

```
## 워크플로우 목록 (2개)

| # | 이름 | 설명 | 버전 | 최종 수정 |
|---|------|------|------|-----------|
| 1 | code-review | PR 코드 리뷰 자동화 | 1.0.0 | 2025-01-06 |
| 2 | deploy-check | 배포 전 점검 워크플로우 | 1.2.0 | 2025-01-05 |

### 사용 가능한 명령어
- 실행: `/workflow-run <이름>`
- 편집: `/workflow-edit <이름>`
- 삭제: `/workflow-delete <이름>`
- 생성: `/workflow-create`
```
